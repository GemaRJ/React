## 🎯 Origen y Motivación
Este proyecto nace de una trayectoria profesional en el sector bancario y el deseo de trasladar esos conocimientos de lógica financiera al desarrollo de aplicaciones modernas. La herramienta no es solo un gestor de datos, sino un ecosistema construido bajo estándares de rigor contable y seguridad visual, diseñado para perfiles Senior, Familiares y Estudiantes.

## 🛠️ Stack Tecnológico
* **Frontend:** React / Next.js / TypeScript.
* **Diseño y Estructura:** Bootstrap (Sistema de rejilla responsivo).
* **Backend & Datos:** Preparado para escalabilidad con Supabase.
* **Infraestructura:** Desplegado en Vercel con integración continua.
* **Interacción:** SweetAlert2 para la gestión de diálogos y notificaciones profesionales.

## 📚 Objetivos de Aprendizaje Alcanzados (RA)
* **RA2 (Sintaxis):** Implementación de lógica modular, funciones flecha y gestión de estados complejos con `useState`.
* **RA3 (Objetos):** Uso avanzado del objeto `Date` para cálculos dinámicos de cierre y de la API `Intl` para precisión monetaria internacional.
* **RA5 (Eventos):** Control de flujos asíncronos (`fetch`), promesas y validación de formularios interactivos con feedback inmediato.
* **RA6 (DOM):** Manipulación dinámica del DOM mediante React y diseño adaptativo basado en el estado de la aplicación.
